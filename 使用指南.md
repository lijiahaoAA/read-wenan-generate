# 小红书文案生成器 - 使用指南

## 快速开始

### 第一步：安装依赖

双击运行 `install.bat` 文件，或在命令行中执行：

```bash
pip install -r requirements.txt
```

### 第二步：启动服务器

双击运行 `start.bat` 文件，或在命令行中执行：

```bash
python app.py
```

看到以下信息表示启动成功：

```
* Running on http://127.0.0.1:5000
```

### 第三步：打开浏览器

在浏览器中访问：`http://localhost:5000`

## 使用流程

1. **输入主题**
   - 在输入框中输入你想要的文案主题
   - 例如：励志、爱情、生活感悟、读书、旅行等

2. **点击生成**
   - 点击"一键生成"按钮
   - 系统会自动完成以下步骤：
     - 生成提示词（约 5-10 秒）
     - 生成 20 条语录（约 10-20 秒）
     - 保存到"文案.md"文件

3. **查看结果**
   - 页面会显示生成的 20 条语录
   - 点击任意语录可以复制到剪贴板
   - 文案会自动保存到项目目录下的"文案.md"文件

4. **继续生成**
   - 可以输入新的主题继续生成
   - 新生成的文案会追加到"文案.md"文件末尾

## 文案文件说明

生成的文案保存在 `文案.md` 文件中，格式如下：

```markdown
## 励志
**生成时间**: 2024-01-01 12:00:00

1. 第一条语录
2. 第二条语录
...
20. 第二十条语录

---

## 爱情
**生成时间**: 2024-01-01 12:05:00

1. 第一条语录
...
```

每次生成都会追加到文件末尾，不会覆盖之前的内容。

## 常见问题

### Q: 生成失败怎么办？

A: 请检查：
- 网络连接是否正常
- `.env` 文件中的 API 密钥是否正确
- DeepSeek API 服务是否正常

### Q: 如何修改 API 密钥？

A: 编辑项目根目录下的 `.env` 文件，修改 `DEEPSEEK_API_KEY` 的值。

### Q: 生成的语录数量不够 20 条？

A: 系统会自动验证语录数量，如果不足 20 条会提示错误。可以重试生成。

### Q: 如何清空历史文案？

A: 直接删除或清空 `文案.md` 文件即可。

### Q: 可以在手机上使用吗？

A: 可以！在同一网络下，使用手机浏览器访问电脑的 IP 地址和端口（例如：`http://192.168.1.100:5000`）。
